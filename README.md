# mac

Mac setup and configuration.

To use:

1. Install Git:

    ```bash
    xcode-select --install
    ```

2. Install setup:

    ```bash
    git clone https://github.com/willfaught/mac ~/Developer/mac
    ```

3. Setup Mac:

    ```bash
    ~/Developer/mac/setup --verbose
    ```

4. Upgrade Mac:

    ```bash
    ~/Developer/mac/setup --verbose --upgrade
    ```
