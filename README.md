# setup

Mac setup and configuration.

To use:

1. Install Git:

    ```bash
    xcode-select --install
    ```

2. Install Setup:

    ```bash
    git clone https://github.com/willfaught/setup ~/Developer/setup
    ```

3. Setup Mac:

    ```bash
    ~/Developer/setup/setup --verbose
    ```

4. Update Mac:

    ```bash
    ~/Developer/setup/setup --verbose --update
    ```
