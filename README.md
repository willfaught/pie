# mac

Mac setup and configuration.

To use:

1. Install Git:

    ```bash
    xcode-select --install
    ```

2. Install mac:

    ```bash
    git clone https://github.com/willfaught/mac ~/Developer/mac
    ```

3. Setup mac:

    ```bash
    ~/Developer/mac/setup
    ```

4. Update mac:

    ```bash
    ~/Developer/mac/setup -u
    ```
